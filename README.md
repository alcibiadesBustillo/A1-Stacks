# Stack Implementation in Python

## Overview

In this assignment, you will implement a basic **Stack** data structure in Python.

A stack follows the **LIFO** principle:

> Last In, First Out

This means that the last element added to the stack must be the first element removed.

For example:

```text
push("A")
push("B")
push("C")

pop() -> "C"
pop() -> "B"
pop() -> "A"
